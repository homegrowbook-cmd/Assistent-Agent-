# Example Images

This directory should contain example plant images for testing and demonstration purposes.

## Recommended Images

For best results with the AI analysis:

1. **High Quality**: Use images with at least 1920x1080 resolution
2. **Good Lighting**: Natural or full-spectrum lighting preferred
3. **Clear Focus**: Sharp focus on the plant, minimal background blur
4. **Full View**: Include the full plant or the specific area being tracked
5. **Consistent Angle**: Use the same angle for progress tracking

## Image Types to Include

### 1. Vegetative Stage
- Full plant overview
- Close-up of leaves
- Top-down canopy view

### 2. Flowering Stage
- Full plant with visible flowers
- Close-up of flower clusters
- Trichome close-ups (macro photography)

### 3. Stress Examples
- Light stress (bleaching)
- Nutrient deficiency
- Overwatering symptoms
- Pest damage

## File Naming Convention

Use descriptive names:
- `plant-001-day-001-overview.jpg`
- `plant-001-day-030-flower-closeup.jpg`
- `plant-002-day-015-leaf-detail.jpg`

## Copyright Notice

Ensure you have the rights to use any images uploaded here. Do not include:
- Copyrighted images without permission
- Images from other sources without attribution
- Personal identifying information

## Adding Test Images

To add test images to this directory:

```bash
# Copy your image
cp /path/to/your/image.jpg public/examples/

# The image will be included in the build automatically
```

## Using Example Images

Example images can be loaded in the application by:
1. Downloading from the `/examples/` directory
2. Using the image upload interface
3. Testing the analysis functionality
