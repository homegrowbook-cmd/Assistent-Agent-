# Image Uploads Directory

Place your plant images here for analysis.

## Supported Formats
- JPG/JPEG
- PNG

## Recommendations
- Use good lighting
- Take photos from consistent angles
- Include a reference object for scale
- Capture the whole plant when possible
- Take close-ups of specific areas of interest

## What Happens Next
1. Images placed here will be automatically processed
2. Analysis results will be generated
3. Processed images will be moved to `images/analyzed/`
4. Posts will be generated based on the analysis

## File Naming Tips
- Use descriptive names: `day_14_plant1.jpg`
- Include dates: `2025-11-21_morning.jpg`
- Specify plant ID: `plant_a_week3.jpg`
